## 3. 제목 작성 패턴

### 유형별 분류

**A. 비유/은유형** (가장 많음)
- "왕관을 벗어던진 버거킹, 비효율의 극치가 만든 의외의 반격"
- "숟가락을 얹지 말고, 파도를 타라"
- "티끌 모아 태산, 금 1g에 담긴 불안의 해답"

**B. 질문형**
- "명동의 약국은 왜 쇼핑 코스가 되었을까?"
- "두쫀쿠가 떠난 자리, 왜 양푼이가 왔을까?"
- "대표님, 우리 막내도 스레드 시킬까요?"

**C. 대비/반전형**
- "'팔아야 할 제품'을 '사고 싶은 제품'으로 바꾸는 전략가의 마케팅"
- "5천 원 도시락 먹으면서 5만 원 수건을 사는 사람들의 비밀"
- "못난이에서 아까운으로: 가치를 재정의하는 언어의 힘"

**D. 선언/도발형**
- "레터 안 읽었지? 그럼 내가 찾아갈게."
- "여백도 전략이 된다."
- "하이네켄은 맥주를 팔지 않는다, '우정'을 판다."

### 제목 작성 원칙
1. 50자 이내 권장
2. 브랜드명/기업명/트렌드 키워드 포함
3. 마케팅 인사이트를 함축하는 한 줄 메시지
4. 일상적 소재 → 전략적 해석의 gap을 만들어 클릭 유도
5. '티끌레터'는 제목에 포함하지 않음 (카테고리로 구분)

## 4. 해시태그 생성 원칙 (신규 제안)

현재 39개 글 전체에 해시태그가 없음. SEO 및 블로그 내 검색 강화를 위해 다음 체계를 제안:

### 고정 태그 (매 글 공통) — 3개
```
#티끌레터  #WK마케팅그룹  #마케팅인사이트
```

### 주제 태그 (글별 2~3개) — 본문 내용 기반 자동 생성

| 카테고리 | 태그 예시 |
|---------|----------|
| 브랜드/기업명 | #버거킹 #듀오링고 #삼성갤럭시 #하이네켄 #롯데리아 #아마존 |
| 마케팅 전략 개념 | #브랜드전략 #포지셔닝 #리브랜딩 #CX전략 #세일즈브랜딩 |
| 마케팅 기법 | #콘텐츠마케팅 #바이럴마케팅 #인플루언서마케팅 #퍼포먼스마케팅 |
| 산업/카테고리 | #식품마케팅 #뷰티트렌드 #리테일 #B2B마케팅 #이커머스 |
| 트렌드 키워드 | #AI마케팅 #숏폼 #MZ세대 #D2C #제로클릭 |

    # 2. 본문에서 마케팅 개념 키워드 매칭

### 태그 수 가이드
- 최소 5개, 최대 10개

## 6. 톤 & 스타일

- **문체**: 경어체 (~습니다, ~입니다) — 전문적이면서 친근
- **시점**: 마케터에게 말하는 2인칭 ("마케터 여러분", "당신의 브랜드는")
- **길이**: 본문 800~1,500자 (읽기 3~5분)
- **이미지**: 대표이미지 1장 중심, 텍스트 위주 구성
- **HTML 사용 안함**: SE3 에디터의 일반 텍스트 입력 방식
